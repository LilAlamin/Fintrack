# Fintrack - Personal Finance Manager

Fintrack is a modern, elegant, and powerful personal finance tracking application built with Flutter. It helps you manage your money, track expenses, and reach your financial goals with a premium user interface.

![App Screenshot](https://raw.githubusercontent.com/LilAlamin/Fintrack/main/assets/images/banner.png) *(Note: Add your actual banner image here)*

## ✨ Features

- **📊 Comprehensive Dashboard**: Get a birds-eye view of your financial health, including monthly budgets and real-time income/expense summaries.
- **👛 Multiple Wallet Management**: Manage different funding sources like Bank Accounts, Cash, and Digital Wallets in one place.
- **📜 Smart Transaction History**: Detailed history with powerful filtering by date range (7 days, 30 days), transaction type, and categories.
- **🌐 Multi-Language Support**: Seamlessly toggle between **English** and **Indonesian** throughout the entire app.
- **🎨 Premium UI/UX**: Elegant design featuring smooth gradients, modern typography (Google Fonts), and intuitive navigation.
- **💾 Local Storage**: Your data stays on your device using high-performance SQLite database.
- **☁️ Persistent Preferences**: Remembers your language and theme choices using SharedPreferences.

## 🛠️ Tech Stack

- **Framework**: [Flutter](https://flutter.dev)
- **Database**: [sqflite](https://pub.dev/packages/sqflite) (SQLite)
- **State/Prefs**: [shared_preferences](https://pub.dev/packages/shared_preferences)
- **Typography**: [google_fonts](https://pub.dev/packages/google_fonts)
- **Localization**: [flutter_localizations](https://api.flutter.dev/flutter/flutter_localizations/flutter_localizations-library.html)
- **Icons**: Material Design Icons

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (Latest stable version)
- Android Studio / VS Code
- Android/iOS Emulator or Physical Device

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/LilAlamin/Fintrack.git
   cd Fintrack
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the app**
   ```bash
   flutter run
   ```

4. **Build Release APK**
   ```bash
   flutter build apk --release --no-tree-shake-icons
   ```
